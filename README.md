## Project setup

### Install NPM
```
npm install
```

### Compile and minifies for development
```
npm run start
```

### Run migrations
```
sequelize db:migrate
```

### Undo migrations
```
sequelize db:migrate:undo:all
```

### Run seeders
```
sequelize db:seed:all
```

### Undo seeders
```
sequelize db:seed:undo:all
```

### Run tests script
```
npm run test
```

### Swagger API documentation
```
http://127.0.0.1:{port}/api-docs
```
